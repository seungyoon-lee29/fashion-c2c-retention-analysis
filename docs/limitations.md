# 한계 (편향과 그 방향)

- **좌측절단(가입 미관측)**: MerRec엔 회원가입 필드가 없어 t0=첫 *관측* 이벤트. 관측 전 활동 유저가 섞임 →
  완화: 지연진입(delayed entry) + `t0≥data_start+buffer`. **잔여 편향 방향**: 베테랑 유저 일부가 신규로 오분류 → 효과 희석 가능.
- **정보적 검열**: 이탈과 전환은 경쟁관계. 이탈을 비정보 검열로 두면 전환 위험 과대추정 → **경쟁위험 모델로 해결**.
- **선택/우검열**: 관측창 끝 근처 진입 유저는 horizon 미관측 → 제외(선택 편향). **방향**: 빠른 전환자 과표집 가능.
- **교란(관측 데이터)**: g-formula/IPTW는 *순차적 교환가능성*(미관측 시변 교란 없음)에 기댐 — 검증 불가 가정.
  → **E-value**로 "이 효과를 없애려면 미관측 교란이 얼마나 강해야 하나"를 정량 봉투. 인과 단정 금지.
- **positivity**: 처치 중첩이 얇으면 IPTW 가중 폭발 → clip + 극단 비율 점검. 실패 시 인과 주장 철회, 연관+E-value만.
- **짧은 horizon(1개월 샘플)**: 리텐션은 수주 범위로 한정. 장기 churn 미관측.
- **마르코프 근사**: 무기억 가정은 시변이력 의존을 무시 → *근사*로만 제시(헤드라인 아님).
- **estimand 미끄러움**: "지표 X를 올린다"의 do(X)는 X가 잘 정의된 조작가능 개입일 때만 의미 → consistency/SUTVA + Goodhart 주석.
- **A/B 부재**: MerRec엔 무작위 실험이 없음 → 모든 인과 추정은 *조건부*. 후속으로 A/B 검증할 가설로 명시.
