# 후보 A — 통합 단일 정본 (v2 비전 기준)
> **세션 추천 × 인과 검증 × 오프폴리시 평가 — 패션 C2C 행동로그 DA 포트폴리오**

## 정체성
"추천을 만들되 그것을 **헤드라인 지표(CTR·Recall)**가 아니라 **북극성(행동 리텐션=재방문)**으로 검증하는
그로스/추천 분석가. 시퀀스 추천을 만들고(1막), 추천이 타깃하는 행동이 리텐션을 *인과적으로* 올리는지 따지고(2막),
추천 정책을 로그만으로 평가하는 OPE를 합성으로 검증한다(3막)."

→ **DA 헤드라인**: 분석가 직무엔 **2막(인과 검증/식별 지도)**을 앞세우고, 1막은 "추천도 만들 줄 안다"는 폭,
3막은 차별화. 생존/g-computation/OPE 머신은 **방법론 깊이**로 둔다.

## 데이터 (단일 출처)
MerRec(Mercari C2C 행동로그) 5파티션·277만 이벤트·43k 유저·49만 세션·31일.
퍼널 도달률 view 98.7 → like 52.6 → cart 22.9 → offer 17.1 → **buy 4.4%**(전환 극희소).
멀티세션 72%·멀티데이 69%(리텐션 estimand 성립). `product_id`=brand×c2 버킷(item 73% 1회성 → 추천 콜드스타트 벽).

## 3막 구조

### 1막 — 세션 시퀀스 추천 (스파인·정체성)
- prefix → next-action(5클래스) + next-item(product_id/c1) 2-헤드. POP→마르코프→Session-KNN→GRU4Rec.
- 시간외 분할(세션 시작시각), leave-last-out. 지표 = Recall@K·NDCG·MRR (인기 대비 *상대개선* 헤드라인, 절대 Recall 금지).
- 🔬 실측: POP→Markov c1 0.59→0.85, product 0.10→0.47 = **"세션 직전 신호 > 인기"**. item-level은 "C2C 콜드스타트 연구"로.

### 2막 — 인과 검증 (식별 가능성 지도, v1 자산 재배치) ← **DA 헤드라인**
- 처치 X = 초기 윈도우 cart≥1 등 관측 레버. Y = embargo 이후 재방문(리텐션 프록시).
- 머신 = v1 이산시간 경쟁위험 해저드(`survival.py`) + g-formula/IPTW/MSM(`impact.py`) + E-value 봉투.
- 🔬 **식별 지도**: 레버 4종(cart·offer·탐색폭·like≥3) **전부 positivity 실패**(e_max=1.0, SMD 0.37–1.21).
  like≥3 = 교과서급 함정(naive +25.7pp가 IPTW 후에도 +25.0pp → 순진하면 "강건 인과"로 오독, 실제론 보정 무의미).
- **결론**: "이 데이터로 식별 가능한 리텐션 레버는 없다 ⇒ A/B 필수." 허영지표 적발 = 정직한 음성. "안 속는 분석가".

### 3막 — 오프폴리시 평가 (적용 한계의 증명, Tier 2)
- 합성(`_synth.py` 확장): 로깅정책 π₀ 설계 → propensity 알려짐 → IPS/SNIPS/DR 무편향 복원 **증명**(편향·분산 곡선).
- 실데이터: 유기적 로그 → 노출 propensity 부재 → 고전 IPS **식별 불가**의 *증명*(BC-π̂₀ 민감도 발산). 진짜 평가는 A/B → W2 이관.

## DA-legible 산출물 매핑 (머신 → 비즈니스 언어)
| 막 | DS 머신 | DA 헤드라인 산출물 |
|---|---|---|
| 1막 | GRU4Rec/마르코프 | 다음행동·카테고리 예측 lift, 콜드스타트 한계 정량화 |
| 2막 | 경쟁위험 생존 + g-comp | **카트 허영지표(Goodhart) + A/B 필수 결론** + (퍼널·코호트는 서술 동반자) |
| 3막 | IPS/DR OPE | "온라인 A/B 없이 정책평가" 역량 시연 |

## 스코프·마일스톤
- **MVS = Phase0 게이트 → 1막 + 2막**(멈추는 선). 3막 OPE는 Tier 2 확장.
- M0 통합/게이트 → M1 1막 베이스라인 → M2 GRU4Rec → M3 2막 식별지도 → M4 3막 OPE → M5 통합 리포트.
- 1개월 제약: W=7·G=2·H=14·lapse=7(합 23<31일). SASRec·3막 실데이터 sweep은 Tier 2.
